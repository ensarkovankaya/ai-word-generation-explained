How AI Predicts the Next Word

1x

What's happening?

Large Language Models (like ChatGPT) generate text one word at a time. At each step, the model calculates a probability for every possible next word, then randomly samples one.

This Galton board visualizes that process:

1 The model starts with an initial word — the prompt you give it.
2 A random number (0–1) is generated. This simulates the probability sampling the model does internally.
3 If the number is < 0.5, the ball falls left. If ≥ 0.5, it falls right — just like the model picking between candidate words.
4 This repeats at each level. Every choice narrows the possibilities, building the sentence word by word.
5 The final path through the tree is the generated sentence. A different set of random numbers → a different sentence.

In real LLMs, the tree has millions of branches and the probabilities are learned from vast amounts of text — but the core idea is the same.
